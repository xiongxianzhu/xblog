# 中文博客成稿

## 结构

1. 一级标题作为文章标题（单个 `#`）。
2. 2～4 个二级小节，每节 2～4 段。
3. 需要时可使用无序/有序列表，避免过长段落。

## 内容

- 围绕用户给出的主题与大纲展开，不跑题。
- 观点具体，避免空洞套话与堆砌形容词。
- 不虚构外链、研究数据、名人引言。

## 输出

- 只输出 Markdown 正文，不要摘要说明或写作过程。
